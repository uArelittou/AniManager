#  AniManager

AniManager 是一款基于 Flask + Vue 3 构建的轻量级本地番剧管理工具。它能够智能扫描你的本地目录下的番剧，对接 Bangumi API自动获取精美海报与剧集信息...的本地番剧管理工具

> **⚠️ 注意：** 你的本地番剧文件夹名请尽量使用规范的中文/日文/英语（无法识别以罗马音命名的文件夹！！！），以确保对接 Bangumi 数据库时能正确精准的识别对应番剧。尽量目标文件夹下只有动画番剧的文件夹，以保持界面美观

---

## ✨ 核心特性 (Features)

- **🤖 智能识别:** 接入 Bangumi API，一键匹配并拉取番剧海报、番剧官方名称。
- **🖥️原生呼出:** 打破浏览器沙盒，点击网页剧集即可直接唤醒电脑本地的默认播放器（推荐 PotPlayer）。
- **📦 单文件运行:** 后端由 PyInstaller 打包为单一 `.exe`，无需配置 Python 环境，双击即用。
- **💅 沉浸式 UI:** 采用 Vue 3 + Tailwind CSS 的暗黑玻璃拟物风简约UI界面，带来优良的观感。
- **🛡️ 隐私优先:** 纯本地化管理，生成的配置文件安全储存于系统 AppData 专属目录，不污染桌面与源码。

## 🚀 快速开始 (Quick Start)

### 方案 A：普通用户（推荐）

如果你只想使用本软件，无需懂任何代码：

1. 前往右侧的 [Releases](https://github.com/你的GitHub用户名/AniManager/releases) 页面。
2. 下载最新版本的 `AniManager.exe`。
3. 双击运行，在自动弹出的浏览器界面中点击右上角 **[+] 导入本地文件夹** 即可开始使用！

### 方案 B：开发者 (For Developers)

如果你想基于源码进行二次开发：

```bash
# 1. 克隆仓库
git clone [https://github.com/你的GitHub用户名/AniManager.git](https://github.com/你的GitHub用户名/AniManager.git)
cd AniManager

# 2. 安装核心依赖
pip install flask requests

# 3. 运行服务
python app.py
```
